# Accommodation Images

Drop accommodation photos here using these exact filenames:

| Filename | Accommodation |
|---|---|
| `druk-lodge.jpg` | Druk Lodge (Paro) |
| `jomolhari-inn.jpg` | Jomolhari Base Camp Inn (Paro) |
| `thimphu-hostel.jpg` | Thimphu Trekkers Hostel (Thimphu) |
| `phobjikha-lodge.jpg` | Phobjikha Crane Lodge (Wangdue Phodrang) |
| `punakha-resort.jpg` | Punakha River Resort (Punakha) |
| `bumthang-farmhouse.jpg` | Bumthang Heritage Farmhouse (Bumthang) |
| `gasa-springs.jpg` | Gasa Hot Springs Camp (Gasa) |
| `yaksa-guesthouse.jpg` | Yaksa Trail Guesthouse (Paro) |

**Recommended specs:** JPEG, 1200×800px, under 500 KB.

After dropping images here, run `docker compose up -d --build frontend` so nginx picks them up.

If an image is missing, the card shows a 🏨 fallback icon.
