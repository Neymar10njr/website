# Hero background image

Drop a file here at:

```
frontend/assets/hero-bg.jpg
```

**Recommended specs:**
- JPEG, 1920×1080 (or wider) — landscape
- Under 800 KB (compress with [tinypng.com](https://tinypng.com) or similar)
- A wide Bhutan landscape: mountains, monasteries, valleys, prayer flags etc.

Until you drop the file, the hero falls back to a beautiful gradient (dark blue → orange) so the page still looks polished.

After dropping the file, run:

```
docker compose up -d --build frontend
```

then hard-refresh the browser.
