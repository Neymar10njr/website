# Trek Images

Drop trek photos here using these exact filenames:

| Filename | Trek |
|---|---|
| `druk-path.jpg` | Druk Path Trek |
| `jomolhari.jpg` | Jomolhari Trek |
| `bumdrak.jpg` | Bumdrak Trek |
| `phobjikha.jpg` | Phobjikha Valley Trek |
| `dagala.jpg` | Dagala Thousand Lakes Trek |
| `snowman.jpg` | Snowman Trek |
| `punakha-winter.jpg` | Punakha Winter Trek |
| `bumthang.jpg` | Bumthang Cultural Trek |
| `dochula.jpg` | Dochula to Lungchutse Trek |
| `laya-gasa.jpg` | Laya Gasa Trek |

**Recommended specs:** JPEG, 1200×800px (3:2 ratio), under 500 KB.

After dropping images here, refresh the browser — no rebuild needed (the frontend container reads from this folder via the bind mount of the build context, but if changes don't appear, run `docker compose up -d --build frontend`).

If an image is missing, the card will show a 🏔️ icon on a gradient background as a fallback.
